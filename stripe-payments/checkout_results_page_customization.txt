Checkout Results Page Customization

Let's say you're selling several eBooks and want to have a customized checkout result page for one of them. Here is the shortcode to display our "Buy now" button:

[accept_stripe_payment name="My eBook" url="http://example.com/my_eBook.pdf" button_text="Buy now!" checkout_url="https://example.com/checkout-result"]

Notice we've used "checkout_url" parameter to specify our custom checkout results page.

Now you want to customize content that is displayed after success payment. For this, you need to enclose it inside [accept_stripe_payment_checkout]. 

For example, if you wanna customize the results for your eBook purcahse, you need to put following on your checkout page:

[accept_stripe_payment_checkout]
Hey, you just purchased {item_name} for {item_price} {currency_code}!
<a href="{item_url}">Click here</a> to download it.
[/accept_stripe_payment_checkout]

Those tags in brackets {} will be automatically replaced with corresponding data.
You can use following tags:

{item_name}
{item_url}
{item_quantity}
{item_price}
{currency_code}
{payer_email}
{transaction_id}
{charge_description}
{billing_address}
{shipping_address}

To customize error message on payment failure, you can use following shortcode: [accept_stripe_payment_checkout_error]
Example:

[accept_stripe_payment_checkout_error]
Oops, something went wrong!
<span style="color: red"><b>{error_msg}</b></span>
[/accept_stripe_payment_checkout_error]

There is only one tag available, which containts the actual error message - {error_msg}
It should be put on the same checkout results page. If error occured, only content inside [accept_stripe_payment_checkout_error] shortcode will be displayed.
If payment was successful, only content inside [accept_stripe_payment_checkout] will be visible for the user.

So our resulting custom checkout page content would look like this:

[accept_stripe_payment_checkout]
Hey, you just purchased {item_name} for {item_price} {currency_code}!
<a href="{item_url}">Click here</a> to download it.
[/accept_stripe_payment_checkout]
[accept_stripe_payment_checkout_error]
Oops, something went wrong!
<span style="color: red"><b>{error_msg}</b></span>
[/accept_stripe_payment_checkout_error]
